# Pizza Ordering System

## Project Overview
The Pizza Ordering System is a Java-based web application designed to simulate an online pizza ordering process. The system allows users to select pizzas, customize options, and generate order details. The application demonstrates the use of Java backend development, REST APIs, and MVC architecture to manage business logic and order processing.

This project highlights the implementation of Object-Oriented Programming principles, layered architecture, and backend service development.

---

## Features
- Select different pizza types
- Customize pizza orders
- Generate order summary and billing
- Manage order processing logic
- Modular architecture using Java backend components

---

## Application Flow
1. User selects pizza type from the menu  
2. User customizes order details  
3. Controller processes the request  
4. Service layer applies business logic  
5. Order details are processed and stored  
6. System generates the final order summary
