# Pizza Ordering System

A Java-based pizza ordering application that allows users to select pizzas, customize toppings, and generate order bills.

## Features
- Select pizza types
- Add toppings
- Generate order bill
- Order summary

## Technologies Used
- Java
- JSP
- Servlets
- JDBC
- MVC Architecture
- SpringBoot
